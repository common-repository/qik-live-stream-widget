== Description ==

A quick plugin I wrote that adds your Qik stream to your sidebar.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload all contents to a directory in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your qik username in Settings->Qik Settings
4. Add the widget to your sidebar from Appearence->widgets
